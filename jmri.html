Using with JMRI

JMRI users on macOS who wish to leverage AppleScript need to perform the following tasks to use this:

  1. Download the version 1.0 of JMRI AppleScript Support to the Downloads folder
  2. Open Terminal.app and enter the following commands, pressing the Enter key after each command:
    1. mkdir -p ~/Library/Preferences/JMRI/lib
    2. mv ~/Downloads/applescript-1.0.jar ~/Library/Preferences/JMRI/lib
    3. cp -n /Applications/JMRI/jmri.conf ~/Library/Preferences/JMRI/jmri.conf
    4. open -e ~/Library/Preferences/JMRI/jmri.conf

  3. Edit jmri.conf to change default_options="" reads:

    default_options="--cp:a=~/Library/Preferences/JMRI/lib/applescript-1.0.jar"
  4. Save jmri.conf. When DecoderPro or PanelPro next starts, it will be able to use AppleScript. This can be verified by selecting the menu item Actions->Run Script… (in DecoderPro) or Panels->Run Script… (in PanelPro) and seeing that AppleScript Files is an option in the File Format pop-up menu.
